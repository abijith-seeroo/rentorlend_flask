A Minimal Application

A minimal Flask application looks something like this:


from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello_world():
    return "<p>Hello, World!</p>"

This is a simple Flask app where the route "/" returns a "Hello, World!" message. You can run this app by saving it in a Python file and running it using python filename.py.
